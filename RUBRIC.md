# AI Code Output Rubric – Slot Machine App

## 1. Product

**Poor**  
App is broken or incomplete. Core slot machine behavior missing or non-functional. No clear AI/token theme.

**Below Expectations**  
Basic functionality exists but is inconsistent or confusing. UI is rough or unclear. AI humor/theme is minimal, forced, or not integrated.

**Meets Expectations**  
Functional slot machine with clear spin/win/lose loop. UI is usable and understandable. AI/token theme is present and adds some personality.

**Exceeds Expectations**  
Polished, responsive app with smooth interactions. Clear feedback (animations, sounds, or messages). AI/token theme is creative, consistent, and genuinely enhances the experience.

---

## 2. Code

**Poor**  
Disorganized, hard to read, or non-working code. Misuse of HTML/CSS/JS. No structure or separation of concerns.

**Below Expectations**  
Works but messy. Limited structure (e.g., everything in one file or function). Naming and organization are inconsistent.

**Meets Expectations**  
Clean, readable code with reasonable structure (HTML/CSS/JS separation). Functions are modular. Logic is understandable.

**Exceeds Expectations**  
Well-structured and maintainable. Clear separation of concerns, good naming, reusable components/functions. Thoughtful use of platform APIs and efficient logic.
